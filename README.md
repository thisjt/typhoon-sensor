# Typhoon Sensor Home Assistant Integration

## Overview

The Typhoon Sensor is a custom Home Assistant integration that provides a single sensor with a fixed value of `24`. This integration is designed to help you automate actions in your Home Assistant setup.

## Features

-   **Fixed Value Sensor**: Emits a single sensor with a constant value of `24`.

## Installation

1. Copy the `typhoon_sensor` folder to your Home Assistant `custom_components` directory.
2. Restart Home Assistant.
3. Add the integration via the Home Assistant UI.

## Configuration

No configuration is required. The integration will automatically create a single sensor with a fixed value of `24`.
